# Arcade · Early experiments with SVG, Web Components and Apollo Server

A distributed arcade game engine and app
& restoration of a Pacman Arcade Cocktail table — Karateco 1984.

# Status

Early stage of development.

# Installation

```bash
npm install
```

# Starting

```bash
npm start
```

Alternatively:

```bash
cd engine && npm start &
cd app && npm start
```
